# Compass Connect

**About**

Compass Connect - Independent, non-medical concierge service supporting NZ & AU patients travelling overseas for surgery.

This project contains everything you need to run the Compass Connect website locally.

**Edit the code in your local development environment**

Any changes pushed to the repo will be reflected in your deployed application.

**Prerequisites:** 

1. Clone the repository using the project's Git URL 
2. Navigate to the project directory
3. Install dependencies: `npm install`

**Run the app: `npm run dev`

**Build for production**

Run `npm run build` to create a production-ready build in the `dist/` folder.

**Deploy**

Deploy the `dist/` folder to any web hosting service (Vercel, Netlify, AWS S3, etc.).

**Development**

- Development server: `npm run dev`
- Build production: `npm run build`
- Lint code: `npm run lint`
